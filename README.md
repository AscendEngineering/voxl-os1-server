# voxl-os1-server

## Installation
1. Ensure you have a breakout board plugged into the voxl2 that will allow you to connect a SD card into the system
2. Ensure the SD Card is formatted as an ext4 format with the name VOXL2
3. Plug in SD card into voxl2 connect + hat
4. ADB into the voxl2 via usb-c connection from parent computer
5. Run "./build.sh"
6. Run "./run.sh"
